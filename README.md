# My Notes

A notes app allow you to create sheets containing tasks/items.

Everything is written in C# with Xamarin.

(Tested carefully on Android only)

## Features

- Sheet: Create, Remove, Pin, Unpin.
- Item: Create, Remove, Pin, Unpin, Complete.
- Theme: Dark, Light.
